> **SkyCast** is a sleek, responsive, and feature-rich weather application built with **Vanilla JavaScript** and **Tailwind CSS**. It provides real-time weather updates, 5-day forecasts, and a premium user experience using **Glassmorphism** design principles.

## âœ¨ Key Features

- **Real-time Data**: Get instant weather updates including temperature, humidity, wind speed, and pressure.
- **5-Day Forecast**: Plan ahead with a detailed daily forecast for the upcoming week.
- **Astro Insights**: Track accurate Sunrise and Sunset timings for any location.
- **Smart Search History**: Automatically saves your recent searches using `localStorage` for quick access.
- **Geolocation Support**: One-tap weather updates based on your current GPS location.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile screens using Tailwind's container-queries.
- **Interactive UI**: Features a sticky glassmorphic header, fade-in animations, and dynamic weather icons.

## âš™ï¸ Installation & Setup

Follow these simple steps getting started:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Mohammad-Farhan-Shaikh/skycast-weather-app.git
    ```

2.  **Get your API Key**:
    -   Sign up at [WeatherAPI.com](https://www.weatherapi.com/).
    -   Copy your free API Key.

3.  **Configure the app**:
    -   Open `script.js`.
    -   Replace `'add-your-api-key-here'` with your actual API key on line 66.
    ```javascript
    const apiKey = "YOUR_API_KEY_HERE";
    ```

4.  **Run the app**:
    -   Simply open `index.html` in your favorite browser.
